# Unwritten Messages Entropy Detection

This example demonstrates how information entropy can be used to detect 'unwritten messages' or hidden patterns within seemingly random data. It generates truly random data, data with a subtle character bias, and data with an embedded repeating pattern, then calculates and compares their Shannon entropy values. Lower entropy indicates more predictability or structure, suggesting the presence of an underlying 'message' not intentionally encoded by a sender.

`python`

## How to Run
